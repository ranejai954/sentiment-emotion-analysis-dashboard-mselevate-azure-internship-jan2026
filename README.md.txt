# 🧠 AI Sentiment Analysis Pro

A sleek, real-time web application that analyzes the sentiment of text using the **VADER** model. It provides a compound score, a visual breakdown of emotions, and a word cloud of key terms.

## 🚀 Features
* **Real-time Analysis**: Get instant Positive, Negative, or Neutral labels.
* **Data Visualization**: Interactive bar charts powered by Plotly.
* **Word Cloud**: Visual representation of word frequency.
* **Modern UI**: Clean, responsive design that supports Dark and Light modes.
