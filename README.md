# sentiment-emotion-analysis-dashboard-mselevate-azure-internship-jan2026
🚀 An AI-powered Natural Language Processing dashboard built for the Microsoft Elevate AICTE Internship. Uses VADER and NRCLex to provide real-time sentiment polarity and granular emotion spectrum analysis.

📝 Overview
This project is an advanced Natural Language Processing (NLP) tool designed to move beyond simple positive/negative sentiment analysis. By leveraging dual-engine processing, the application provides a comprehensive "Emotion Spectrum," detecting specific human affects such as Joy, Trust, Fear, Surprise, and Sadness.

✨ Key Features
- Dual-Engine Analysis: Uses VADER for sentiment intensity and NRCLex for multi-class emotion mapping.
- Real-Time Processing: Instant analysis of user-provided text.
- Interactive Visualizations: Dynamic bar charts and donut charts powered by Plotly.
- User-Friendly UI: A clean, responsive dashboard built with Streamlit.

🛠️ Tech Stack
1. Language: Python 3.9+
2. NLP Libraries: NLTK, vaderSentiment, NRCLex, TextBlob
3. UI/Deployment: Streamlit, Streamlit Community Cloud & Local Host
4. Data Viz: Plotly Express, WordCloud, Matplotlib

🚀 How to Run ?
1. Clone the repository:
git clone https://github.com/ranejai954/sentiment-emotion-analysis-dashboard-mselevate-azure-internship-jan2026.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the application:
streamlit run app.py
